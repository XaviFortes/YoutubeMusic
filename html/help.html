Help

This is a not official Youtube Music Player, made for Windows. It's easy to use, interactive, works with any windows PC, MacOS and Linux. It is fast, secure, etc.

You can use some shortcuts like :

Ctrl + K to hide

Ctrl + I to show

F11 to toggle the FullScreen

and media shortcuts, when you are in app  you can use the media keys like play/pause, next track, previous track, change volume and many more.

If you want the open source you can find it https://github.com/XaviFortes/YoutubeMusic

You can post any issue in https://github.com/XaviFortes/YoutubeMusic/issues
